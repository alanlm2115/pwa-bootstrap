## CSS Framework
- **Elegido:** Bootstrap 5 (latest)
- **Cómo se integra:**
  - `npm i bootstrap @popperjs/core`
  - `src/styles.scss`: `@import "bootstrap/scss/bootstrap";`
  - `src/main.ts`: `import 'bootstrap';`
